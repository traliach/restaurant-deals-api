# restaurant-deals-api (Backend)

## Overview
Node + Express + TypeScript REST API for Restaurant Deals.

## Tech Stack
- Node.js, Express, TypeScript
- MongoDB + Mongoose
- JWT Authentication
- Stripe (payments)

## Running locally
```bash
npm install
npm run dev
```

## Environment variables
Create a `.env` file:
- PORT=
- MONGO_URI=
- JWT_SECRET=
- STRIPE_SECRET_KEY=
